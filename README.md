# 密码管理系统

## 简介

这是我大学课程设计,  使用python的Django框架编写的一个前后端不分离的项目(前端使用bootstrap),   由于我经常记不住密码, 因此就想设计如此一个网站来保存密码

## 使用

### 启动项目

```shell
python manage.py runserver
```
